# Changelog

All notable changes to **Umarell** will be documented in this file.

## [0.1.0] — 2026-05-30

### Added
- Initial public release
- Live construction site webcam panel in the sidebar
- 6 preset Italian construction sites (Wikimedia Commons hotlinks)
- Custom webcam support (snapshot JPEG/PNG URLs)
- Animated SVG umarell with flat cap, moustache and idle sway
- 24 bilingual (EN / IT) random mutterings
- Comment language setting: `auto` · `en` · `it` · `both`
- Evening mode (19:00–06:00) — swaps the photo for a construction GIF, badge turns from `LIVE` to `🌙 NIGHT`
- Configurable refresh interval, comment interval, evening window, evening GIF URL
- Bilingual UI strings everywhere (commands, popups, tooltips)
